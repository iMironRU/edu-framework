[![Язык: русский](https://img.shields.io/badge/Язык-русский-blue)]()

Открытый отраслевой стандарт для управления образовательными организациями России. Покрывает ВУЗ, СПО, ДО, ДПО и аспирантуру. Включает нормативную базу, лучшие практики, описание данных и машиночитаемые форматы НПА.

> **Статус.** Активная разработка. Каркас и первые домены готовы. PR приветствуются — см. [CONTRIBUTING.md](CONTRIBUTING.md).

---

## Зачем этот фреймворк

В большинстве российских ОО управленческие процессы либо не описаны вовсе, либо описаны разрозненно — без связи между собой, без привязки к актуальной нормативной базе, без учёта того, как данные переходят из одного процесса в другой. Этот фреймворк решает три задачи:

1. **Полнота** — 13 доменов покрывают все аспекты деятельности ОО: от лицензирования до воспитательной работы.
2. **Связность** — каждый домен знает, от чего он зависит и что питает. Граф зависимостей строится автоматически.
3. **Применимость** — маркеры типа ОО (ВУЗ / СПО / ДО / ДПО / Асп) показывают, что обязательно для вас, что опционально.

## Откуда начать

### Открываю новую ОО или выстраиваю процессы с нуля
Начните с лицензирования — домен [D02](domains/D02-licensing/) описывает требования,
пакет документов и процедуру целиком. Нормативная база — в [D01](domains/D01-normative/).

### Готовлюсь к конкретному событию

| Событие | Куда идти |
|---|---|
| Получение или переоформление лицензии | [D02 → D02-P01](domains/D02-licensing/processes/D02-P01.md) |
| Государственная аккредитация | [D02 → D02-P02](domains/D02-licensing/processes/D02-P02.md) |
| Приёмная кампания | [D05 → D05-P01](domains/D05-contingent/processes/D05-P01.md) |
| Зачисление обучающегося | [D05 → D05-P02](domains/D05-contingent/processes/D05-P02.md) |
| Мониторинг изменений законодательства | [D01 → D01-P01](domains/D01-normative/processes/D01-P01.md) |

## Машиночитаемые стандарты

Раздел [`standards/`](standards/) содержит НПА в структурированных форматах:
- **ФГОС в JSON** — JSON Schema + примеры. Используется для валидации учебных планов, формирования ОПОП, интеграции с ИС.
- **ФЗ-273** — структурированные выдержки с привязкой к доменам.
