# ФГОС в машиночитаемом формате

Этот раздел содержит Федеральные государственные образовательные стандарты
в формате JSON — для использования в ИС, скриптах валидации и автоматической
генерации документов (ОПОП, учебные планы, матрицы компетенций).

---

## Зачем это нужно

ФГОС в PDF — документ для чтения. ФГОС в JSON — источник данных для:
- автоматической проверки соответствия ОПОП требованиям стандарта
- генерации матрицы компетенций
- валидации рабочих программ дисциплин
- импорта в ИС при создании новой образовательной программы
- сравнения версий стандартов при актуализации программ

```

    ├── FGOS-VO-44.03.04-2022.json   ← пример: Педагогическое образование
    └── FGOS-SPO-09.02.07-2020.json  ← пример: Информационные системы и программирование
```

## Именование файлов

Формат: `FGOS-<ТИП>-<КОД_УГСН>-<ГОД>.json`

| Часть | Значение | Пример |
|---|---|---|
| ТИП | VO / SPO / OO / DPO | `VO` |
| КОД_УГСН | Код направления через точки | `44.03.04` |
| ГОД | Год приказа об утверждении | `2022` |

Пример полного имени: `FGOS-VO-44.03.04-2022.json`

---

## Схема (fgos.schema.json)

Схема описывает четыре обязательных блока:

| Блок | Содержание |
|---|---|
| `meta` | Реквизиты: тип, уровень, код УГСН, номер и дата приказа |
| `program` | Характеристики программы: квалификация, формы, сроки, объём в з.е. |
| `competencies` | УК, ОПК, ПК — код + формулировка |
| `structure` | Блоки программы с объёмом в з.е. |

Дополнительный блок `requirements` — кадровые и МТБ-требования (опционально).

## Валидация

```bash
# Установить зависимости
pip install jsonschema --break-system-packages

# Валидировать один файл
python tools/validate-fgos.py standards/fgos/data/FGOS-VO-44.03.04-2022.json

# Валидировать все файлы
python tools/validate-fgos.py --all
```

## Как добавить новый ФГОС

1. Откройте официальный текст стандарта на [fgosvo.ru](https://fgosvo.ru) или [consultant.ru](https://consultant.ru)
2. Скопируйте `data/_EXAMPLE.json` как шаблон
3. Заполните все обязательные блоки по тексту стандарта
4. Запустите валидацию
5. Откройте PR — один ФГОС = один PR

**Важно:** проверяйте актуальность стандарта. Некоторые ФГОС имеют изменения,
вносимые отдельными приказами. Указывайте в `meta.source_url` ссылку на
актуальную редакцию, в `meta.order_date` — дату последних изменений.

## Связи с доменами фреймворка

| Домен | Как использует ФГОС |
|---|---|
| D01 Нормативная база | Ссылается на стандарт как НПА |
| D02 Лицензирование | Код УГСН из `meta.ugsn_code` нужен для лицензии |
| D04 Образовательная деятельность | Объём программы, формы обучения |
| D06 УМД и контент | Компетенции — основа для матрицы РПД |
