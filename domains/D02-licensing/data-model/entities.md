# D02 · Модель данных

## Сущности домена

### Лицензия

| Поле | Тип | Описание |
|---|---|---|
| `номер` | string | Регистрационный номер лицензии |
| `дата_выдачи` | date | Дата выдачи лицензии |
| `лицензирующий_орган` | string | Рободрнадзор или орган субъекта РФ |
| `статус` | enum | `действует` / `переоформляется` / `приостановлена` / `аннулирована` |
| `приложения` | list[ПриложениеКЛицензии] | Перечень лицензированных программ и адресов |
| `url_реестр` | url | Ссылка на запись в федеральном реестре лицензий |

### ПриложениеКЛицензии

| Поле | Тип | Описание |
|---|---|---|
| `номер_приложения` | string | Порядковый номер приложения |
| `адрес` | string | Адрес места осуществления деятельности |
| `программы` | list[ЛицензируемаяПрограмма] | Программы по данному адресу |

### ЛицензируемаяПрограмма

| Поле | Тип | Описание |
|---|---|---|
| `уровень_образования` | enum | НОО / ООО / СОО / СПО / ВО / ДО / ДПО и др. |
| `код_оквэд` | string | Код по ОКВЭД (напр. 85.13) |
| `наименование_программы` | string? | Для ВО/СПО — наименование направления |
| `код_специальности` | string? | Для ВО/СПО — код ФГОС |

### СвидетельствоОбАккредитации

| Поле | Тип | Описание |
|---|---|---|
| `регистрационный_номер` | string | Номер свидетельства |
| `дата_выдачи` | date | Дата выдачи |
| `срок_действия_до` | date | Дата окончания срока действия |
| `аккредитованные_программы` | list[string] | Перечень аккредитованных программ |
| `статус` | enum | `действует` / `приостановлена` / `лишена` |

### ДелоЛицензиата

Агрегат, объединяющий все документы лицензионного цикла одной ОО.

| Поле | Тип | Описание |
|---|---|---|
| `лицензия` | Лицензия | Текущая лицензия |
| `история_заявлений` | list[Заявление] | Все поданные заявления с датами и статусами |
| `акты_проверок` | list[АктПроверки] | Результаты плановых и внеплановых проверок |
| `заключения_надзорных_органов` | list[Заключение] | МЧС, Роспотребнадзор (с датой истечения) |
| `отчёты_самообследования` | list[ОтчётОСамообследовании] | Ежегодные отчёты |
| `свидетельства_аккредитации` | list[СвидетельствоОбАккредитации] | Все свидетельства |

## ERD (концептуальная схема)

```
ОО  1──────────────────────1  ДелоЛицензиата
                              │
                              ├──1 Лицензия ──* ПриложениеКЛицензии ──* ЛицензируемаяПрограмма
                              ├──* СвидетельствоОбАккредитации
                              ├──* АктПроверки
                              └──* Заключение (МЧС / Роспотребнадзор)
```
