# D00: Название домена

> **Кластер:** Название кластера · **Статус:** draft · **Версия:** 0.1.0

## Что это и зачем выделен отдельно

2–3 предложения: что охватывает домен, почему он выделен самостоятельно,
а не является частью другого домена. Без канцелярита.

## Что входит в домен

- Процесс А
- Процесс Б
- Процесс В

## Что не входит (частые заблуждения)

- X — это домен D0Y, потому что...
- Y — это сквозная функция D13

## Нормативная база (ключевые акты)

- **ФЗ-273, ст. N** — что регулирует применительно к домену
- **Приказ № XXX** — описание

Полный список → [metadata.yml](metadata.yml) → `regulatory_refs`

## Связи с другими доменами

- **Зависит от:** D01 (нормативная база задаёт ограничения)
- **Питает:** D13 (является источником данных для отчётности)
- **Взаимодействует с:** D0X (обмен данными)
