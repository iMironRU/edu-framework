## Активные проблемы (v3.5.0)

### [KI-001] String256Type — непоследовательный maxLength

**Обнаружено:** v3.4.0 · **Статус:** не исправлена в v3.5.0

| Файл | maxLength | Ожидалось |
|---|---|---|
| `DespatchXsd/EpguPaidContractResult/EpguPaidContractResult.xsd` | 1024 | 256 |
| `DespatchXsd/EpguPaidContractData/EpguPaidContractData.xsd` | 255 | 256 |

**Что делать:** использовать значение из конкретного файла схемы. Валидировать XML локально.
