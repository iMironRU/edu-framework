# D09-P01: Заключение договора на платные образовательные услуги

**Домен:** D09 · **Применимость:** ВУЗ, СПО, ДО, ДПО, Аспирантура (при платном обучении)
**Статус:** draft

## Назначение

Зафиксировать обязательства сторон до зачисления обучающегося: какая услуга оказывается,
по какой цене, с какой разбивкой по периодам. Договор — предусловие зачисления:
зачисление невозможно без заключённого договора (при этом договор может быть ещё не
оплачен).

## Нормативное основание

- ФЗ-273, ст. 54 — договор об образовании, существенные условия.
- ФЗ-273, ст. 101 — платная образовательная деятельность.
- Постановление Правительства РФ от 15.09.2020 № 1441 — Правила оказания платных
  образовательных услуг (заказчик/обучающийся, форма договора). ⚠ действует до 31.12.2026.

## Входы

- Образовательная услуга с действующей ценой по периодам (прейскурант).
- Действующая версия шаблона договора.
- Сведения о сторонах: обучающийся, заказчик, плательщик(и).
- Применимая категория обучающегося (при наличии льготы).

## Шаги

1. Определить услугу и собрать график по периодам из действующих периодных цен по нарезке
   из учётной политики (полная сумма — итог графика, не вводится вручную).
2. Применить модификатор категории по правилу политики (если льгота в цене договора).
3. Зафиксировать стороны договора (обучающийся / заказчик / плательщик[и]).
4. Зафиксировать слепок действующей версии шаблона и дату начала обучения.
5. Создать договор как версию плана v1; договор переходит в статус «Заключён».

## Выходы / артефакты

- Договор в статусе «Заключён» (версия плана v1 со снимком графика).
- Неизменяемый слепок версии шаблона на момент подписания.

## Участники

| Роль | Действие |
|---|---|
| Договорной отдел | Оформляет договор, фиксирует стороны |
| Планово-финансовое управление | Подтверждает цену и график по прейскуранту |
| Обучающийся / заказчик | Подписывает договор |

## Типовые ошибки

- Ввод полной суммы вручную вместо сборки из периодных цен — расхождение с прейскурантом.
- Заказчик приравнивается к обучающемуся, хотя это разные лица — некорректная первичка.
- Зачисление оформляется до заключения договора — нарушение предусловия.
